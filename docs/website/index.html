Asterisk CDR Statistics

Introduction

Asterisk CDR Statistics (AGCDR) is a PHP application designed to work with data generated by the Asterisk MySQL CDR add-on. It produces periodical reports on the usage of your Asterisk PBX and uses Google Charts to produce dynamic visualisations of call volumes. AGCDR supports multiple database servers allowing you to monitor all the Asterisk servers for which you are responsible from one installation of AGCDR.

This project is in beta stage and is subject to further amendments and improvements before a formal version 1 release. I welcome all bug reports and suggestions for improvement. The package is released under the GNU General Public License v3.0.

Pre-requisites

AGCDR requires the following to run. This list is pretty standard, but please check it carefully.

  • Apache 2.2.x server with virtual hosts and mod_rewrite enabled.
  • PHP 5.3.3. This is what I've tested it on. It might work on older versions, it might not. It definitely won't work on PHP 4.x. Your PHP installation will require the MySQL and cURL modules to be compiled.
  • Access to a CDR database server used by a PBX that uses cdr_mysql.
